# See No Evil: Semantic Context-Aware Privacy Risk Detection for AR

## Introduction
This is the official repository of Paper: See No Evil: Semantic Context-Aware Privacy Risk Detection for AR. The paper is accepted to IEEE ICASSP 2026. It contains download our collected datasets and codes.

## Required packages


This project was built based on unity's template of AR mobile, and used the editor version of unity: 2022.3.6f1.
